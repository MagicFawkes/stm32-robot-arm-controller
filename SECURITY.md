# Security Policy

## Purpose

This project is an educational embedded firmware project for robotic arm control.
While it is not intended for safety-critical or production use, responsible handling of potential software or system vulnerabilities is encouraged.

## Reporting a Vulnerability

If you discover a security issue, safety risk, or critical malfunction related to this project, please report it by contacting the maintainer.

Please include:

* a clear description of the issue
* steps required to reproduce the problem
* potential impact on system behaviour or safety
* suggested mitigation if available

Do not publicly disclose critical vulnerabilities before the maintainer has had reasonable time to review and respond.

## Scope

Security considerations may include, but are not limited to:

* unintended servo movement or unsafe control behaviour
* communication or protocol weaknesses
* incorrect handling of emergency stop conditions
* configuration errors that may lead to hazardous operation
* software defects affecting reliable real-time control

## Disclaimer

This firmware is provided for educational and experimental purposes only.
It is not designed, tested or certified for use in safety-critical, industrial, medical or commercial environments.

Use of this software and associated hardware is entirely at the user’s own risk.

## Supported Versions

This repository currently provides development firmware only.
No long-term security support or guaranteed update policy is provided.
