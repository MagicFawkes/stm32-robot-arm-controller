# Contributing

Thank you for your interest in this project.

Contributions such as bug reports, improvements or technical suggestions are welcome.

## Reporting Issues

Please use the provided issue templates when reporting problems.

Include clear technical information and, if possible, steps required to reproduce the behaviour.

## Pull Requests

When submitting a pull request:

* describe the change clearly
* explain the motivation
* keep the modification focused
* ensure that the change was reasonably tested

## Project Scope

This repository contains an educational robotics control project.

Contributions should fit the existing architecture and overall technical direction of the system.

## Discussion of Larger Changes

For significant modifications or new features, please open an issue first to discuss the idea.

## Contribution Notice

All contributions are voluntary and provided without any form of compensation or obligation.

This project is developed in spare time and maintained on a best-effort basis.
No paid support, guaranteed response time or implementation commitment can be provided.

By contributing to this repository, you agree that your contribution may be reviewed, modified or declined at the discretion of the project maintainer.
