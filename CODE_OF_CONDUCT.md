# Code of Conduct

## Purpose

This project aims to provide a professional and respectful environment for technical discussion, collaboration and knowledge exchange related to embedded systems, robotics and software development.

## Expected Behaviour

Participants are expected to:

* communicate in a respectful and constructive manner
* focus discussions on technical topics and project-related matters
* provide helpful feedback and suggestions
* respect different experience levels and viewpoints
* use clear and professional language

## Unacceptable Behaviour

The following behaviour is not tolerated:

* personal attacks, harassment or discrimination
* offensive, abusive or insulting language
* trolling, provocation or deliberate disruption of discussions
* spam or irrelevant content
* sharing harmful or unsafe instructions without proper context

## Enforcement

The project maintainer reserves the right to moderate discussions, close issues or remove contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies to all interactions within this repository, including issues, pull requests, discussions and other communication channels related to the project.

## Contact

If you experience or observe unacceptable behaviour, please contact the maintainer.

---

This project is primarily an educational and experimental embedded systems project.
Constructive technical collaboration is welcome.
